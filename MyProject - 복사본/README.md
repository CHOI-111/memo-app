# 메모장 앱

회원가입, 로그인 기능이 있는 메모장 웹 애플리케이션입니다.

## 기능

- ✅ 회원가입 및 로그인 (로컬 스토리지 기반)
- ✅ 사용자별 메모 관리
- ✅ 메모 생성, 수정, 삭제
- ✅ 메모 검색
- ✅ 서버 없이 작동 (브라우저에서 바로 실행 가능)
- ✅ 로컬 스토리지에 데이터 저장

## 설치 및 실행

### 간단한 실행 방법 (권장)

**서버 설치나 설정 없이 바로 사용 가능합니다!**

1. `index.html` 파일을 브라우저로 열기 (더블클릭)
2. 회원가입 또는 로그인
3. 메모 작성 시작!

### 온라인에서 사용하기

**GitHub Pages로 배포된 버전 사용:**
- [배포 가이드 보기](DEPLOY.md)
- GitHub Pages를 통해 무료로 웹에 배포할 수 있습니다

### 서버 없이 작동

이 앱은 **서버 없이** 브라우저에서 바로 작동합니다:
- 모든 데이터는 브라우저의 로컬 스토리지에 저장됩니다
- 회원가입/로그인 정보도 로컬 스토리지에 저장됩니다
- 인터넷 연결 없이도 작동합니다

### 서버를 사용하고 싶은 경우 (선택사항)

서버를 사용하려면:

```bash
cd server
npm install
npm start
```

그 후 `http://localhost:3001`로 접속하세요.

## 사용 방법

1. **회원가입**: 처음 사용하는 경우 회원가입을 진행하세요.
2. **로그인**: 사용자명(또는 이메일)과 비밀번호로 로그인하세요.
3. **메모 작성**: 로그인 후 "새 메모" 버튼을 클릭하여 메모를 작성하세요.
4. **메모 관리**: 메모를 수정하거나 삭제할 수 있습니다.
5. **검색**: 상단 검색창에서 메모를 검색할 수 있습니다.

## 기술 스택

- **프론트엔드**: HTML, CSS (Tailwind CSS), JavaScript
- **데이터 저장**: 브라우저 로컬 스토리지
- **서버**: 없음 (서버 없이 작동)

## 데이터 저장

- 모든 데이터는 브라우저의 **로컬 스토리지**에 저장됩니다
- 사용자 정보: `app_users` 키에 저장
- 메모 정보: `app_memos` 키에 저장
- 각 사용자의 메모는 `user_id`로 구분됩니다

## 문제 해결

### 회원가입/로그인이 작동하지 않는 경우

1. **브라우저 콘솔 확인**
   - F12를 눌러 개발자 도구 열기
   - Console 탭에서 오류 메시지 확인

2. **로컬 스토리지 확인**
   - 개발자 도구 → Application 탭 → Local Storage
   - `app_users`와 `app_memos` 키가 있는지 확인

3. **브라우저 호환성**
   - 최신 브라우저 사용 권장 (Chrome, Edge, Firefox 등)
   - 로컬 스토리지를 지원하는 브라우저 필요

### 데이터 초기화

로컬 스토리지를 초기화하려면:
1. F12로 개발자 도구 열기
2. Application 탭 → Local Storage
3. 해당 사이트의 로컬 스토리지 항목 삭제
